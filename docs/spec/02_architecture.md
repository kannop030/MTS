## コンポーネント構成

### Gradio Web UI
- ファイルアップロード
- 処理オプション選択
- 進捗表示
- 結果プレビューとダウンロード

### FastAPI Backend
- `/api/upload` : ファイル受け取り
- `/api/process` : 処理実行（非同期）
- `/api/status/{job_id}` : 進捗確認
- `/api/download/{job_id}` : 結果取得

### Transcriber（文字起こし）
- 動画から音声を抽出（FFmpeg）
- faster-whisperで文字起こし
- タイムスタンプ付きテキストを出力

### Extractor（スライド抽出）
- PySceneDetectでシーン変化を検出
- OpenCVで代表フレームを抽出・保存
- EasyOCRでスライドテキストを認識

### Summarizer（要約・議事録）
- 文字起こし＋スライドテキストを結合
- Ollama APIへプロンプトを送信
- 要約・議事録をMarkdownで出力
