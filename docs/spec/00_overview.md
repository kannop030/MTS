# システム概要

## プロジェクト名
Media Transcriber & Summarizer

## 目的
会議録画・講義動画・音声データから、文字起こし・スライド抽出・要約・議事録生成を
完全ローカル環境で自動化するシステムを構築する。

## 背景
- 会議や講義の録画データを手動でテキスト化する作業が負荷になっている
- 外部サービスへのデータアップロードによる情報漏洩リスクを避けたい
- 無料・ローカルで完結するツールを自社環境に持ちたい

## 対象ユーザー
- 会議録画を議事録にしたい業務ユーザー
- 講義動画を復習用テキストにしたい学習者

## システム方針
| 方針 | 内容 |
|------|------|
| 完全ローカル | データは外部に一切送信しない |
| 無料 | 有償サービス・APIは使用しない |
| GPU任意 | CPU環境でも動作する（GPU時は高速化） |
| Docker対応 | 環境を問わずデプロイ可能 |
| 仕様書駆動 | 仕様を先に定義し、実装はそれに従う |

## 対応入力形式
- 動画: `.mp4` `.mkv` `.avi` `.mov`
- 音声: `.mp3` `.wav` `.m4a` `.ogg` `.flac`
- テキスト: `.txt` `.md`

## アウトプット
| 出力物 | 形式 | 説明 |
|--------|------|------|
| 文字起こし | `.txt` | 全発話テキスト（タイムスタンプ付き） |
| スライド画像 | `.png` | シーン変化時に抽出したフレーム |
| スライドテキスト | `.txt` | OCRで認識したスライド内テキスト |
| 要約 | `.md` | 内容の要点まとめ |
| 議事録 | `.md` | 構造化された議事録（日時・参加者・決定事項・TODO） |
